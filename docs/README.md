# StarkPayHub

**StarkPayHub** is an on-chain subscription payment protocol built on Starknet. It enables any SaaS product to accept recurring USDC payments — without custodians, without banks, and without writing smart contract code.

Payments are processed natively on Starknet using Account Abstraction: users sign once and renewals happen automatically via the keeper bot.

**Live app:** [starkpayhub.vercel.app](https://starkpayhub.vercel.app)

## Why StarkPayHub?

| Traditional Payments | StarkPayHub |
|---|---|
| Stripe takes 2.9% + $0.30 per transaction | Zero platform fees |
| Centralized, can freeze your account | Non-custodial, permissionless |
| Manual integration, lots of boilerplate | 3-line SDK integration |
| Fiat only | USDC on Starknet |
| Auto-renewal via card-on-file | Auto-renewal via keeper bot |
